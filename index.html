เหตุผลที่อยากเรียน Full stack web development

สนใจในการพัฒนาเว็บและต้องการสร้างเว็บไซต์ที่ครบวงจร ซึ่งสามารถจัดการทั้งด้าน Frontend และ Backend ได้อย่างมีประสิทธิภาพ

ความคาดหวังต่อวิชานี้

หวังว่าจะได้รับความรู้ที่ลึกซึ้งเกี่ยวกับการพัฒนาเว็บไซต์ในทุกด้าน และสามารถนำความรู้นั้นไปใช้ในการสร้างโปรเจคจริงได้

อะไรจุดอ่อน ที่ตนเองต้องพัฒนาเกี่ยวกับทักษะ Programming และ แนวทางการพัฒนา

จุดอ่อนของฉันคือการจัดการกับฐานข้อมูลและการเขียนโค้ดที่มีประสิทธิภาพ โดยฉันต้องการฝึกฝนเพิ่มเติมในด้านนี้

Link ไปยังหน้า Website ที่ตนเองชื่นชอบ หรือ ได้รับแรงบันดาลใจ
